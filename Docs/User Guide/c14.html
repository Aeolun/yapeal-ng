Chapter 1. Introduction

1.1. What is Yapeal?

Yapeal is an acronym for Yet Another PHP EVE API Library. It was developed as part of a web base eve tool that has yet to be released but I thought the backend scripts that were used to pull information from the eve API should be extracted into thier own project and made available to others while the rest of the development continued.

Why call it Yet another? First off there's already been a couple other PHP API lib/classes made by other people and follow in the tradition of other open source projects I decide it was time to release what I'd been writing with a name along the lines of YaST (Yet another Setup Tool) or Wine (Wine Is Not an Emulator).
