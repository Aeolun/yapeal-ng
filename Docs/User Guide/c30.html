Chapter 2. Installation

2.1. Getting software

A new Google code project has been setup for Yapeal at http://code.google.com/p/yapeal/ where you can find the latest version plus the latest version of this documention.

For now Yapeal is only being offered through the subversion at Google code. With future releases it maybe offered as tgz and/or zip files. See the project home above for instructions on how to get it from the subversion.
