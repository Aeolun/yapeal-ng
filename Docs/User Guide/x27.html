1.4. What do I need to use it?

Nothing special in the software. You should be able to use it on a shared host with just a little configuration. It does require PHP5 and MySQL. It's been developed on a AMD 64 1GB linux system with PHP version 5.2.6 and MySQL version 5.0.51a. It uses cURL for API connections and now I have replace MDB2 with ADOdb for the Database. If you'd like to use it like I do you'll also need shell access and be able to setup a crontab job. I believe it would also be used on a windows server if you know how to set up something like a Unix type crontab but I haven't tried it nor do I know for sure that it can be done. If someone tries it and would like to send me an explaination of how to do it I'll include it here in the future. Even better if they develop a BAT or CMD wrapper that can be used I'll be glad to include it with Yapeal.
