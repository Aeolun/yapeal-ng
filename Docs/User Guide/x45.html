2.3. Setting up database(s)

Yapeal is being used to pull (request) data from some of the eve and corp APIs and that code is can be considered useable at this time while the code for char APIs is still in testing but largely based on the corp code which has been tested. Code for map and other areas available through the EVE API has not been written because my other projects haven't used them for anything yet but the code hopefully can be added soon if someone would help the author out adding them.

More detailed information about setting up databases to be added later. Sample SQL to build some of the tables has been include but has not been fully checked or tested yet.
